Feedback Survey

This is a survey created by and for Emerging Digital Academy coding bootcamp students. Feedback will be collected over 4 views: How well are you feeling, how are you understanding the content, how well do you feel supported, and any additional comments you might have. Each question is asked on a scale of 1-5 except the comments has an area to write additional text. In a separate review page, the current feedback values are displayed along with a submit button. Once all steps are complete, your app will save the feedback in the database and give you an option to start again and leave more feedback.

Main pages



Extra Admin Capability

The Admin page is a centralized hub where all user feedback is organized in a detailed table. It provides essential information about each feedback entry, including the questions answered, comments, and submission dates. Additionally, administrators can efficiently manage feedback by deleting irrelevant entries, ensuring the quality and relevance of the feedback repository.
