Invoicer

A simpler way to create Invoices for your small business

What is invoicer?

This was my Solo Project from my time at EDA. Invoicer is a web application that simplifies and streamlines the process of customer management, service tracking, and generating professional printable Invoices for a small business. It was modeled after my families small local Lawn Care Company that my Mom is currently running our company today and I made this application to help her/ any non technical user to streamline their invoicing process.

View Project on Github

Main pages



Extra Admin Capability

The Admin page is where you will find the last 3 of the photos above, as well as the next photo. On the main Admin page you will be greeted with a personalized message and see a monthly financial overview of how much money you have generated in invoices each month. This is where the admin user can add or remove services, customers, and view archived invoices. The Admin page is also where employees, as well as company information, can be added or removed


The view is the customer table under the Admin page and it shows how there is ability to search and sort through the customers. There is also a form that pops up to add new customers as well as a table that shows up when you click on the archived customers button. On the customer table itself there is ability to do inline edits and archive customers that you no longer service. When you view the archived customers table there is the ability to un-archive as well to bring those customers back into the active customer list.
